# JetPDF Utility 🚀

A lightweight desktop PDF tool built with Python.

## ✨ Features
- Merge PDFs
- Extract Pages
- Extract Text
- Images to PDF
- PDF to Excel
- PDF to Word
- Drag & Drop support

## 🖥️ Tech Stack
- Python
- CustomTkinter
- pdfplumber
- tkinterdnd2

## 📦 Build
Packaged using PyInstaller into a single executable.

## 🚀 Usage
Run the `.exe` file (no installation required)

## 📌 Note
Designed for office environments with restricted installations.
